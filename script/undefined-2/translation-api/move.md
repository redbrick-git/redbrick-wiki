# Move

### 정의

> ### 지정한 방향으로, 입력한 값만큼 오브젝트를 움직입니다.
>
> * **x** \
>   x축 이동량을 입력합니다.
> * **y**\
>   y축 이동량를 입력합니다.
> * **z**\
>   z축 이동량를 입력합니다.
> * **up**\
>   상향 이동량을 입력합니다.\
>   \[ Move ] \[ y ] \[ 양수 ]와 같은 결과가 나타납니다.
> * **down**\
>   하향 이동량을 입력합니다.\
>   \[ Move ] \[ y ] \[ 음수 ]와 같은 결과가 나타납니다.
> * **left**\
>   좌향 이동량을 입력합니다.\
>   \[ Move ] \[ x ] \[ 음수 ]와 같은 결과가 나타납니다.
> * **right**\
>   우향 이동량을 입력합니다.\
>   \[ Move ] \[ x ] \[ 양수 ]와 같은 결과가 나타납니다.
