# Rotate

### 정의

> ### 지정한 방향으로, 입력한 값만큼 오브젝트를 바로 회전합니다.
>
> * **x** \
>   x축 회전량을 입력합니다.
> * **y**\
>   y축 회전량를 입력합니다.
> * **z**\
>   z축 회전량를 입력합니다.
> * **up**\
>   \[ Rotate ] \[ y ] \[ 양수 ]와 같은 결과가 나타납니다.
> * **down**\
>   \[ Rotate ] \[ y ] \[ 음수 ]와 같은 결과가 나타납니다.
> * **left**\
>   \[ Rotate ] \[ x ] \[ 음수 ]와 같은 결과가 나타납니다.
> * **right**\
>   \[ Rotate ] \[ x ] \[ 양수 ]와 같은 결과가 나타납니다.
