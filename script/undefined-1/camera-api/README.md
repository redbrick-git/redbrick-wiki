# Camera API

### Overview

|             API Format            | Return Value | 기능                                   |
| :-------------------------------: | :----------: | ------------------------------------ |
| setCameraProperty(fov, near, far) |       -      | 카메라의 시야각, near plane,far planer값을 조절 |
|       setZoomRange(min, max)      |       -      | 카메라의 최소, 최대 줌 제한을 조절                 |
