# Timer API

### Overview

|   API Format  | Return Value | 기능                   |
| :-----------: | :----------: | -------------------- |
|  startTimer() |       -      | 호출 시 타이머를 시작         |
|  pauseTimer() |       -      | 호출 시 타이머를 일시정지       |
| resumeTimer() |       -      | 호출 시 타이머를 멈춘 시점부터 시작 |
|  resetTimer() |       -      | 호출 시 타이머를 리셋         |
