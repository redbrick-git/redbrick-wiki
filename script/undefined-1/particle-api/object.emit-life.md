# object.emit(life)

### 정의

> ### 지정한 시간만큼 파티클 오브젝트의 파티클을 방출합니다.
>
> * **life**\
>   시간을 초 단위로 입력합니다.



### 예시

```javascript
const particle = getObject("ParticleSystem(751)")
particle.emit(3)
```
