# object.findFirstAncestor()

### 정의

> ### 오브젝트의 부모 오브젝트를 반환합니다.
>
> * 레드브릭의 오브젝트는 계층 관계를 형성합니다.
> * 선택한 오브젝트의 부모 오브젝트를 반환합니다.



### 예시

```javascript
const tree = getObject("nature_palmtree_001(cdf)")
const treeParent = tree.findFirstAncestor()
```
