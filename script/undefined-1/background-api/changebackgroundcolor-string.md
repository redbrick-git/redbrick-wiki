# changeBackgroundColor(string)

### 정의

> ### 지정한 색으로 배경의 색을 변경합니다.
>
> * **string**\
>   색 이름 혹은 16진수 컬러 코드를 입력합니다.



### 예시

```javascript
onClickEmpty(function () {
    changeBackgroundColor("#FFB291")
})
```
